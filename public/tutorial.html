The Manufacturer is making decisions based on old, distorted information.
This causes the classic Bullwhip Effect — small demand changes create huge swings upstream.

Every order is a trade-off.

You must balance three competing costs — every single round.

📉
Lost Sales
$4 / unit
Order too little → you run out → customer walks away → lost sale
🗑️
